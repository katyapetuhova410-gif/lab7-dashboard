# Офтальмологический дашборд

Лабораторная работа №7 по дисциплине "Программирование"

## Вариант
- **Вариант:** №39
- **Предметная область:** Офтальмология

## Описание
Интерактивный дашборд для анализа офтальмологических данных (внутриглазное давление, толщина роговицы, острота зрения).

## Функциональность
- 4 типа графиков: линейный, столбчатый, точечный, тепловая карта
- Фильтрация по пациенту, категории риска, времени суток
- Экспорт графиков в PNG/PDF

## Технологии
- Python 3.12
- Tkinter (GUI)
- Matplotlib / Seaborn (графики)
- Pandas / NumPy (обработка данных)

## Установка и запуск
```bash
pip install matplotlib seaborn pandas numpy
python dashboard.py
